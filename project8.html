Expense Tracker App

  • Source code

This is an android app meant to help people analyze their income and expenses while also tracking their past transactions

Some of the features of the application include :

  • Completely Offline
  • Income Analysis
  • Transaction Analysis
  • Expense Analsysis
  • Dark/Light Mode

Technologies Used include :

  • Jetpack Compose
  • MVVM architecture
  • Firebase App Distribution - Distribute new builds to testers
  • Dagger Hilt for Dependency Injection
  • Room Database to store data
  • Firebase Crashlytics - Report crashes that occur in the app
