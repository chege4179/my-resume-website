Built a React-Native chat application called GlacierChat using Javascript

  • Live Demo
  • Source code

I built a one-to-one chat application using React-Native for both Android and iOS. The application backend is purely backed by Firebase.The messages are stored on cloud firestore in firebase. I used firestore to leverage the realtime functionality that comes with firebase cloud firestore

Some of the features of the application include :

  • Google sign-in using your email account
  • One-to-one chat rooms
  • Real-time message syncing -

Libaraies Used include :

  • React Native Gifted Chat - Easy to implement Chat UI for react native apps
  • React Navigation - For navigation in react native apps
  • React Native Vector Icons - Icons for react native apps
  • React Native Firebase - Wrappper package for react native apps to interat with firebase SDKs
