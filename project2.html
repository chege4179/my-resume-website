Built a dynamic blogging platform using Next.js and Express.js

  • Live Demo
  • Source code

This is a blogging platform built using Next.js and Express.js for the server with Mongo DB for the backend

Some of the features of the web application include :

  • Comment System
  • Ability to follow and be follwed by creators
  • The app is available as a PWA(Progressive Web App) so it can installed as a mobile app or desktop application
  • Rich Text Editor to enable a good experience
  • Ability to upload images in the text editor
