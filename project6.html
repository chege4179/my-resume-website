Built a native android chat application called PChat using Kotlin

  • Live Demo
  • Source code

Built a native android chat application called PChat using Kotlin. The App uses websockets for real time communication and a server written in Node.js.The server implements socket.io server API for realtime communication and Mongo DB for user information storage and message information storage

Some of the features of the application include :

  • One to one chat messaging
  • Real time communication

Technologies Used include :

  • Jetpack Compose
  • Dagger Hilt for Dependency Injection
  • Retrofit to make Api Calls
  • Socket.io client for kotlin for websocket communication
  • MVVM architecture
